## Описание

JS скрипт для создания дополнительных полей в формах на сайте

Почему решил делать посредством JS

1. Простота использования
   - Необходимо указать вscripts/main.js в переменной this.forms формы в которые необходимо будет добавить поля
2. Быстрое внедрение в другие проекты

## Важно

- В хук создания лида необходимо добавить обработку поля результатов

```php
if (!empty($fields['afcf-result'])) {
    $additionalFields[] = 'Дополнительные товары: ' . $fields['afcf-result'];
}
```
